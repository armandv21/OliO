# OliO — OPES Frontend

Optimisation de Portefeuille — app frontend restructurée.

## Structure
- `index.html` — Shell HTML
- `styles/` — CSS modulaire (variables, layout, composants)
- `js/` — Modules JavaScript (auth, assets, charts, optimization)
- `data/` — Données statiques

## Stack
- Vanilla JS (ES Modules)
- Supabase (auth + BDD)
- Plotly.js (graphiques)
- Google Fonts: Libre Baskerville + DM Sans
